# 영천중앙교회 홈페이지 수정 설계

작성일: 2026-06-25
근거 문서: `[홈페이지]수정사항.pdf` (고객사 전달, 8페이지)

## 배경 / 목표

기존 사이트는 데스크탑에서 좌측 사이드바 네비게이션 + 긴 단일 스크롤(10개 섹션) 구조다.
고객사 요청은 (1) 네비게이션을 상단 우측 가로 메뉴로 환원, (2) 홈을 4스크린으로 압축,
(3) 공식 PCK(예장통합 총회) 휘장 적용, (4) 메뉴 정보구조(IA) 재정비, (5) 푸터·OG 정리다.

## 확정된 결정 사항

- **네비 배치:** 좌측 사이드바 → 상단 우측 가로 메뉴 (모바일 우상단 버거 유지)
- **헤더 바 색:** 솔리드 브랜드 블루(`accent-deep`) + 흰색 메뉴 (결정 A)
- **로고:** 공식 PCK 휘장 사용(자체 로고 미제작). 휘장은 엠블럼 단독(워드마크 없음) → "영천중앙교회 / YEONGCHEONJOONGANG" 텍스트는 자체 조합
- **헤더 로고 처리:** 흰 라운드 배지 + 컬러 휘장 (결정 C). 푸터·밝은 배경은 컬러 휘장 단독
- **워드마크 영문:** `YEONGCHEON CENTRAL` → `YEONGCHEONJOONGANG`, 볼드
- **홈 스크롤:** 4스크린으로 재구성 (PDF 의도)
- **메뉴 IA:** 하이브리드 — 기존 페이지/필터 재사용 + 신규(행복선언·FNQ·예배안내)만 제작

## 작업 항목 (PDF 항목별)

### 1. 네비게이션 — 상단 우측 가로 메뉴
- `Sidebar` 제거/비활성, `Header`의 기존 상단 가로 메뉴를 전 브레이크포인트에서 사용
- 로고 좌측, 메뉴 우측 정렬
- 바 스타일: 솔리드 `accent-deep` 블루 + 흰색 로고·메뉴
- 모바일: 우상단 버거 현행 유지
- 관련 파일: `src/app/layout.tsx`(사이드바 마운트 제거), `src/components/layout/Header.tsx`, `Sidebar.tsx`, `BrandLogo.tsx`, `globals.css`(`.ycc-side` 정리)

### 2. 로고 자산
- 출처: `Downloads/PCK_Logo`(컬러/그레이/단선 + 투명 PNG, .ai/.pdf 벡터), `Downloads/PCK_CI팬톤`(CI 매뉴얼)
- 가능하면 벡터(.pdf/.ai)를 SVG로 변환해 사용(변환 도구 확인 필요), 불가 시 투명 PNG 사용
- `public/brand/`에 배치: 컬러 휘장(밝은 배경용), 흰색/단색 휘장(다크 바용)
- `BrandLogo.tsx`를 PCK 휘장 + 자체 워드마크 텍스트 조합으로 교체
- 파비콘/OG 이미지(`/brand/ycc-icon*`, `/brand/ycc-og.png`)도 PCK 휘장 기반으로 갱신 검토

### 3. 링크 미리보기(OG) 수식어구 제거
- `src/app/layout.tsx` metadata: description/openGraph/twitter에서 "삶의 소망을 주는 은혜로운" 제거 → 교회명 중심 간결화
- OG 이미지: 교회명 + 사진/배경색 형태 점검·갱신

### 4. 홈 4스크린 재구성
1. **히어로** — 현행 `ImmersiveHero` 유지 (문구 검토)
2. **#2 환영 문구** (`Manifesto` 교체) — "오래된 믿음 위에, 새로운 은혜가 머무는 / 영천중앙교회에 오신 여러분 환영합니다", 배경 블루→라벤더 부드러운 세로 그라데이션
3. **#3 소개 + 바로가기** (`FullBleedBand` 교체) — "대한예수교장로회 영천중앙교회는…" 문구, 버튼 [예배안내][오시는길], 배경 교회사진 오버레이 강화(더 흐리게)
4. **#4 진입 버튼 3종** — 말씀 / 주일학교 / 찬양, 버튼 배경 사진, 각 카드 하위 링크
   - 말씀 › 주일예배·주일찬양예배·수요예배 (말씀요약 "한줄요약": 기존 AI 설교요약 첫 문장 재사용)
   - 주일학교 › 유치부·아동부·중고등부
   - 찬양 › 시온찬양대·할렐루야찬양대·호산나찬양대
- 제거 후 메뉴로 이동: Gallery, NextGenCommunity, RecentSermons, Verse, Visit, CtaBand
- `HomeScrollController`를 4스크린에 맞춰 조정, `page.tsx` 구성 변경

안내        › 행복선언(신규) · 주일예배 · 주일학교(유치·아동·중고등부) · 청년부 · 수요예배 · 새벽예배 · 금요기도회
            → 예배 요일·시간·장소를 '예배 안내' 페이지에 섹션/표로 (worship.ts 데이터 재사용)

## 단계 분할

- **Phase 1:** 네비 상단 이동 + 블루 바 + PCK 휘장/워드마크 (로고 자산 처리 포함)
- **Phase 2:** 홈 4스크린 재구성(#2~#4) + 색/그라데이션/오버레이
- **Phase 3:** 메뉴 IA 정비 + 행복선언·FNQ·예배안내 신규 페이지
- **Phase 4:** 푸터(소셜·디모데앱) + OG 수식어구 정리

## 비범위 (YAGNI)

- 사이트 전면 리브랜딩(블루 테마는 유지, 색은 튜닝만)
- 디모데앱 SSO/딥링크 연동(단순 외부 링크만)
- 설교 영상 신규 파이프라인(기존 /sermons·AI요약 재사용)
